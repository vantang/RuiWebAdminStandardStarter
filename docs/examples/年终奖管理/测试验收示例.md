# 年终奖管理测试验收示例

## 1. 测试目标

本次测试用于验证年终奖管理模块首期建设是否满足以下业务目标：

- HR 可以查看并筛选奖金测算结果
- 部门负责人可以对待确认记录执行确认或驳回
- 详情页可以完整展示奖金构成、审批轨迹与异常提示

## 2. 测试范围

本次纳入范围：

- 奖金测算列表
- 奖金详情查看
- 审批确认流程
- 导出任务查询
- 接口文档与技术方案一致性检查

本次不纳入范围：

- 与外部 OA 平台的真实审批回写
- 历史年度数据迁移与对比分析

## 3. 测试环境与前置条件

- 环境：本地开发环境
- 前端地址：`npm run dev:start` 启动后的本地前端地址
- Mock / 后端地址：本地 `mock-server`
- 测试账号：`admin / admin123`
- 前置数据：至少准备 1 条待确认记录、1 条已确认记录、1 条异常记录

## 4. 核心用例

| 编号 | 场景 | 操作步骤 | 预期结果 |
| --- | --- | --- | --- |
| TC-01 | 查询奖金测算列表 | 进入奖金测算页，按部门和状态筛选 | 列表正确刷新，分页与总数正确 |
| TC-02 | 查看奖金详情 | 从列表打开任意一条详情 | 可看到奖金构成、审批轨迹和异常说明 |
| TC-03 | 审批确认 | 对待确认记录执行确认 | 状态更新为已确认，操作按钮消失 |
| TC-04 | 审批驳回 | 对待确认记录执行驳回并填写原因 | 状态更新为待修正，驳回原因可见 |
| TC-05 | 查询导出任务 | 进入导出任务页，按批次和状态筛选 | 列表正确展示导出状态、文件名和创建时间 |

## 使用原则

- 新项目启动前，先复制这些规范，再按项目实际情况最小化调整。
- 如果某条规范需要被突破，必须在项目文档中显式说明原因。
- 模板的目标是降低重复决策，而不是完全限制具体业务实现。
- 如果仓库已经派生为真实业务项目，优先遵守业务事实，再复用模板规范与 AI 协作资产。
